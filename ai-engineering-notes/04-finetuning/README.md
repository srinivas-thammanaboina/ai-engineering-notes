# 04 · Finetuning & Local Models

*~2.5 weeks*

Placed after RAG deliberately — learn when *not* to finetune before learning how. RAG often solves problems people reach for finetuning to fix.

## Topics

- **Finetuning fundamentals** — when it helps, when it doesn't
- **Efficient methods** — LoRA / QLoRA
- **Local models** — running models with Ollama

## Papers

LoRA · PEFT survey (Lialin et al.) · Mixture of Experts (Switch Transformer / Mixtral)

## Project

Fine-tune a small open model with LoRA on a narrow task, run it locally via Ollama, and compare before/after.

## Done when

I can articulate the finetuning-vs-RAG decision and have a working LoRA fine-tune running locally.
